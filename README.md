# iTabs Campaigns

Hosted iTabs for Kickstarter campaigns.

## Structure

```
/campaigns
  /campaign-name-1
    index.html
  /campaign-name-2
    index.html
```

## URLs

Each campaign gets a clean URL:
- `yourusername.github.io/itabs-campaigns/campaigns/smart-backpack`
- `yourusername.github.io/itabs-campaigns/campaigns/cool-gadget`

## How to Add a Campaign

1. Create folder in `/campaigns` with campaign name (use dashes, lowercase)
2. Paste generated iTabs code into `index.html`
3. Commit and push
4. Share the URL

## Future: Custom Domain

When ready, add custom domain:
- `itabs.ai/k/smart-backpack`
- `itabs.ai/k/cool-gadget`
